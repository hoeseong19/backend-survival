# REST API

## REST API

최초는 로이 필딩의 논문

실무에서 사용하는건 엄밀히 말하면 REST API 가 아님

API : 클라이언트와 서버의 통신 메뉴얼

## URI & MIME type

### URI

자원을 식별하는 방법

URL: 위치를 통해서 자원을 식별하는 방법

URN: 고유한 이름을 통해서 자원을 식별하는 방법

### MIME type

자원의 유형을 알려주는 header 속성

## Collection Pattern

자원을 그룹(collection)으로 관리하는 URL 설계 방법

CURD -> POST, PUT, GET, DELETE
